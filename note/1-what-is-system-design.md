# What is System Design?

## Definition

System Design is the process of planning how different parts of a software application work together to solve a problem.

---

## Why do we need it?

- To handle many users
- To improve performance
- To make the application reliable
- To make it scalable

---

## Real-Life Example

Designing a hospital.

Doctors, nurses, reception, pharmacy, and billing all work together.

The hospital is the system.

---

## Key Idea

System Design focuses on the overall architecture, not individual functions or algorithms.

---

## Interview Tip

If someone asks,
"Design YouTube,"
